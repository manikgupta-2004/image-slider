# 📖 About The Project

This project is a modern and fully responsive **Image Slider** built using **HTML5, CSS3, and JavaScript**.

The application provides a smooth image browsing experience with interactive navigation controls, clean animations, and responsive layouts that work seamlessly across desktop, tablet, and mobile devices.

The primary objective of this project was to strengthen frontend development skills, improve JavaScript DOM manipulation knowledge, and practice building reusable UI components.
